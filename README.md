## О проекте

Репозиторий для исследования и сравнения различных AI моделей при решении задач разработки на 1С-Битрикс. Каждая модель получает одинаковое техническое задание и оценивается по единым критериям.

## Цель исследования

Определить сильные и слабые стороны современных AI моделей при:
- Разработке модулей для 1С-Битрикс
- Соблюдении архитектурных принципов D7
- Работе с enterprise-требованиями
- Понимании специфики CMS и нового ядра
- Создании современных публичных компонентов

## Методология

### Базовый промпт
Все модели получают единый системный промпт с базовыми знаниями о разработке на 1С-Битрикс, включающий:
- Архитектурные принципы и лучшие практики D7
- Стандарты кодирования PSR-12/Bitrix Standard
- Работу с ORM, контроллерами, событиями
- Enterprise-требования и безопасность

**[Текст базового промпта →](bitrix_prompt.md)**

### Тестовое задание
Разработка модуля «Избранное» для каталога товаров с требованиями:
- REST API через `\Bitrix\Main\Engine\Controller`
- ORM DataManager для хранения данных
- Шифрованные cookie (`CryptoCookie`) для гостей
- Миграция данных при авторизации
- Кэширование с тегами и инвалидация
- Публичный компонент для страницы товара

**[Полное техническое задание →](bitrix_ai_challenge.md)**

### Критерии оценки (10 критериев × 10 баллов = 100)

| # | Критерий | Что проверяет |
|---|----------|---------------|
| 1 | Базовая функциональность | Установка, добавление/удаление, миграция |
| 2 | ORM и работа с данными | DataManager, валидаторы, связи |
| 3 | REST Controller D7 | `Engine\Controller`, actions, права |
| 4 | Работа с Cookie D7 | `CryptoCookie`, флаги безопасности |
| 5 | События D7 | Регистрация, миграция, инвалидация |
| 6 | Кэширование с тегами | `TaggedCache`, `clearByTag()` |
| 7 | Работа с инфоблоками D7 | `Iblock\Elements`, D7 API |
| 8 | Архитектура и структура | Controller → Service → Repository |
| 9 | Безопасность | Валидация, CSRF, шифрование |
| 10 | Публичный компонент | Дизайн, AJAX, анимации, адаптивность |

**[Подробные критерии оценки →](criteria.md)**
